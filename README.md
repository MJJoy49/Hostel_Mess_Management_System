# Hostel / Mess Management System

The **Hostel / Mess Management System** is a web-based application designed to digitally manage all operations of a hostel or mess. The system allows an admin to create a hostel, manage members, rooms, meals, bills, payments, notifications, and seat advertisements.

---

## Features

### 1. Authentication & Roles
- Admin and Member login  
- Role-based access control  
- Secure session-based authentication  

### 2. Hostel / Mess Management
- Create and manage mess/hostel information  
- Assign an admin to each mess  
- Store address, capacity, and other basic details  

### 3. Member Management
- Add / edit / remove members  
- Manage member profiles (contact, email, profession, etc.)  
- Assign members to rooms  
- Track member status (active / inactive / on leave)  

### 4. Room Management
- Create rooms with:
  - Room number  
  - Capacity  
  - Rent per seat  
  - Facilities  
- Track current occupancy  
- Room-wise member listing  

### 5. Meal & Attendance Management
- Create daily meals with:
  - Date  
  - Meal type (breakfast / lunch / dinner)  
  - Menu details  
- Member meal attendance tracking  
- Manage meal rates (history with effective dates)  

### 6. Billing & Payments
- Monthly bill generation for each member:
  - Total meals  
  - Meal rate  
  - Total amount, paid, and due  
- Record payments:
  - Payment for: meal bill / seat rent / both  
  - Payment method: cash, bKash, Nagad, bank  
  - Transaction ID / note  
- Monthly payment summary and individual history  
- Net balance calculation (deposit vs bill)  

### 7. Daily Bazar & Expenses
- Daily bazar list:
  - Date, items, total amount, bazar-by  
- General expenses:
  - Category-wise expense tracking (e.g., utility, others)  

### 8. Notices & Announcements
- Admin can post announcements/notices for members  
- Notice list with title, message, posted-by, and date  

### 9. Seat Advertisements & Requests
- Create seat advertisements:
  - Vacant seats  
  - Rent per seat  
  - Contact person & number  
  - Mess address & details  
- Public/visitor seat apply form  
- Store and manage incoming seat requests  

---

## Tech Stack

- **Backend:** PHP (custom MVC-style structure)  
- **Frontend:** HTML5, CSS3, JavaScript  
- **Database:** MySQL (`messContro`)  
- **Server Environment:** XAMPP / WAMP / LAMP (Apache + PHP + MySQL)  
- **Version Control:** Git & GitHub
